
# ESP-NOW Framework v6 (Release Candidate)

Scalable ESP32 telemetry framework using ESP-NOW mesh networking.

Main capabilities:

- dynamic mesh routing
- OTA firmware distribution over ESP-NOW
- sensor scheduler
- auto-detection stubs (I2C + ultrasonic)
- MQTT gateway integration
- network diagnostics
- reservoir monitoring example

Supported hardware:

- ESP32
- ESP32-C3
- ESP32-S3
