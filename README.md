
# ESP-NOW Framework v1.0

Experimental framework for building ESP32 sensor networks using ESP-NOW.

Features:

- lightweight binary protocol
- mesh relay capability
- modular sensor drivers
- OTA-ready architecture
- MQTT/Home Assistant gateway support

Supported boards:

- ESP32 DevKit
- ESP32-C3
- ESP32-S3

Compile using Arduino IDE or PlatformIO.
