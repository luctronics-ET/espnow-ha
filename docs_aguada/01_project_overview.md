
# AGUADA – ESP-NOW IoT Platform

## Overview

AGUADA is an ESP32-based IoT platform focused on:

- ESP-NOW mesh networking
- distributed sensor nodes
- low infrastructure dependency
- long distance telemetry
- integration with Home Assistant

Primary use case:

Reservoir monitoring using ultrasonic level sensors connected to ESP32 nodes.

## Key Goals

- Avoid Wi‑Fi infrastructure in the field
- Support long-distance communication (~160 m)
- Use ESP-NOW mesh routing
- OTA firmware updates
- Auto-configurable nodes
- Modular sensor drivers
- Plug‑and‑play architecture similar to Zigbee
