
# Hardware Platform

Supported boards:

ESP32 DevKit
ESP32-C3 SuperMini
ESP32-S3

## Typical Sensor Node

ESP32-C3 SuperMini
Ultrasonic level sensor
Temperature / humidity sensor
Battery or DC supply

## Gateway Hardware

ESP32-S3 preferred

Possible interfaces:

USB
Ethernet (ENC28J60)
Wi‑Fi (optional)

## Node Features

Potential optional peripherals:

OLED display
LED indicators
Buttons
Buzzer or speaker
