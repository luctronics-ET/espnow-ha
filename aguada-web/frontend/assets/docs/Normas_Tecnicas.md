# Normas Técnicas – Sistema Supervisório de Abastecimento de Água

Documento de referência normativa para o Aguada Web.

Este arquivo espelha o conteúdo principal de normas técnicas utilizado no projeto.
A versão de referência completa está em:

- `aguada-web/docs/sistemas-hidricos/Normas_Tecnicas.md`

## Normas-chave

- ABNT NBR 12217
- ABNT NBR 12218
- ABNT NBR 12214
- ABNT NBR 5626
- ABNT NBR ISO 24510 / 24511 / 24512
- Portaria GM/MS nº 2.914/2011

## Regras operacionais críticas

- Bloqueio automático de recalque abaixo do nível mínimo operacional.
- Registro auditável de comandos manuais e automáticos.
- Controle de qualidade da medição (filtro, histerese, validação de leituras).
- Indicadores operacionais e alarmes para suporte à decisão.
