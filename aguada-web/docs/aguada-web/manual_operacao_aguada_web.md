# Manual de Operação — Aguada Web

## Objetivo

Orientar operadores no uso diário das telas de Dashboard, SCADA, Histórico, Consumo, Abastecimento, Dados e Relatórios.

## Fluxo diário sugerido

1. Verificar status online dos reservatórios no Dashboard.
2. Conferir níveis e estados no SCADA.
3. Validar tendências em Histórico/Consumo.
4. Emitir Relatório Diário e revisar aba de Balanço Hídrico.
5. Registrar intervenções manuais com justificativa.

## Boas práticas

- Validar horário e data antes de exportar relatório.
- Confirmar variações abruptas com mais de uma leitura.
- Registrar anomalias e ações corretivas em trilha auditável.
