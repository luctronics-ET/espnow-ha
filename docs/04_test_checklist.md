# Field Testing Checklist

## Radio Tests

Measure RSSI between nodes

Distances: 50 m 100 m 150 m 200 m

Record packet loss.

## Mesh Routing Test

Disable direct link Verify relay forwarding works.

## OTA Test

Upload firmware over mesh Measure update time.

## Sensor Reliability

Test sensor failures: disconnect sensor check alarm generation

## Power Test

Measure consumption battery runtime estimation

## Stress Test

Simulate 10+ nodes sending packets simultaneously.
